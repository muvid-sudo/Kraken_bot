The file is the project description.

Modules:
1) Data collection - obtaining actual data from crypto exchange
- get_current_price();
- get_current_balance();
- get_open_positions();
- get_open_orders();
- get_closed_orders();
- get_previous_orders();
- get_information_orders();
- get_information_open_margin_orders();
- get_information_asset_volume().

2) Data analyzer - preparing and analyse data
2.1) preparing_data;
2.2) sorting_data;
2.3) data analyze;
2.4) record data.

3) Trade strategy - проверка обработанных данных и сигналы
3.1) Описание и расчет стратегии;
3.2) Информация о позициях;
3.3) Отправка сигналов.

4) Decision making - принятие решения о совершении транзакции
4.1) Формирование ордера;
4.2) Сравнение данных;
4.3) Запись данных.

5) Actions - совершения транзакции
- Добавить ордер;
- Отменить ордер;
- Отменить все ордера;
- Отменить все ордера после падения.

6) Position - информация о позиции
- ID asset;
- Наименование позиции по активу;
- Дата совершения сделки;
- Дата закрытия сделки;
- Комиссия с сделки;
- Количество актива;
- Цена актива при открытии сделки;
- Текущая цена актива;
- Цена актива при закрытии сделки;
- Стоимость открытия позиции;
- Текущая стоимость позиции;
- Стоимость закрытия позиции;
- Цена стоп-лосса;
- Цена тейк-профита;
- Маржа;
- Доля в портфеле;
- Финансовый результат;
- Запись позиции в историю.

7) Data output - вывод данных для пользователя
- Вывод данных о позиции;
- Вывод данных о счете.

8) API - подключение к бирже
- Хранит ключи для подключения к бирже;
- Отправляет запрос на подключение к бирже;
- Получает ответ от биржи.

9) Caching - сбор и хранение данных
- Сбор данных о позиции;
- Сбор данных о решениях;
- сбор данных об анализе данных;
